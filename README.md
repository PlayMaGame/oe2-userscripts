# OE2 Userscripts

Userscripts for [Outer Empires 2](https://outerempires.com/). Requires Tampermonkey or Violentmonkey.

## Active

| Script | What it does |
|---|---|
| **afk-warning.user.js** | Flash, beep, and warn when AFK for 15s inside an instance |
| **auto-pilot-panel-centering.user.js** | Center the map info panel at the top of the screen |
| **darkreader.js** | Dark mode theme injector |
| **default-filters.user.js** | Keeps specific Tactical filters always enabled |
| **exit-countdown-beep.user.js** | Slow beep when exit countdown reaches 5-1 |
| **hide-hud.user.js** | Hide HUD elements. F1 toggle, auto-hide on tab switch |
| **hide-lables.user.js** | Cycle Ship/Body labels with F2 |
| **JOB target highlighter.user.js** | Stable ship target highlighting for OE2 jobs |
| **jobs-navigation-qol.user.js** | Colors location/system/bounty references in job text; click to search |
| **local-time.user.js** | Hide location panel and add custom overlay with local time |
| **oe-chat-font-control.user.js** | Adjust in-game chat font size and toggle bold |
| **oe_apdock.user.js** | Adds "AP+Dock" and "AP+Warp" buttons next to Auto Pilot |
| **oe2-bounty-tracker.user.js** | Live target list for active bounty, tracks eliminations from combat log |
| **oe2-combat-hp.user.js** | Live target HP panel + intel (HP, missiles, torpedoes, gear, ranks) |
| **oe2-obs-control.user.js** | Direct WebSocket to obs-websocket — toggles filters on galaxy map |
| **oe2-twitch-chat.user.js** | Embeds Twitch chat as a channel in the in-game Social Media group |
| **oe2-twitch-viewers.user.js** | Twitch viewer count in ui_character_detail |
| **oe2-vital.user.js** | Real-time ship maintenance grade — A-F rating, component dot grid |
| **ping.user.js** | Replace version with color-coded ping, FPS and memory usage |
| **shields-off-warning.user.js** | Warn when shields are OFF while inside an instance |
| **system-explorer-filter.user.js** | Filter/recolor items in the System Explorer panel |
| **zoom-toggler.user.js** | Cycle zoom levels with F3 |
